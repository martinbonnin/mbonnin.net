Dev:

```
npm ci
npm run dev
```

Deploy:

```
npm ci
npm run build
firebase deploy --only hosting
```

Features:

- everything is markdown with codeblocks support
- image resizing
- RSS feed
- mobile UI
- dark mode
- "edit on GitHub"

Missing:

- anchor links
- admonitions
- site map (left column)
- table of contents (right column)
- social card previews
